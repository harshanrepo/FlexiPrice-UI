💳 PriceFlex UI

A clean and responsive pricing plans layout built using pure HTML and CSS. This project showcases a modern card-based pricing section commonly used in SaaS websites.

---

🚀 Demo

🔗 Live Demo: https://harshanrepo.github.io/FlexiPrice-UI/

---

📌 Features

* Responsive flexbox layout
* Clean and minimal UI design
* Interactive hover animations
* Pricing tiers (Basic, Pro, Premium)
* Reusable card structure

---

🛠️ Tech Stack

* HTML5
* CSS3 (Flexbox)
